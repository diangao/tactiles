01 · The gap

Braille text covers words.
Diagrams stay sighted.

Option A

Order a tactile graphic from a service.

Turnaround measured in weeks. Per-diagram fee. Doesn't survive a curriculum revision.

lead time: weeks
Option B

Substitute a Unicode-braille font.

Letters render. Structure does not. Double bonds, rings, stereochemistry collapse into characters.

structure dropped
Option C

Skip the diagram.

Caption the figure with a sentence. The student loses the diagram as a thing to think with.

information lost

A teacher's edit on a chemistry diagram should be one upload, not a procurement workflow.

03 · The thing that broke us

An edit can silently lie.

Before

Acetic acid, faithful render.

CH3—C(=O)—OH

The carbonyl C=O double bond is present. Verifier compares canonical SMILES to the rendered IR.

ready preflight chip
After — same diagram

"Make labels bigger" drops the double bond.

› make labels bigger
CH3—C=O—OH  → C—O

The depiction shifted. A blind student would learn a different molecule. A teacher would not see this.

needs review →

parse
Claude vision, server-side.

Image → canonical SMILES. The API key never reaches the browser — it lives only in the Vercel function env.

verify
rdkit-js, deterministic.

Canonicalize source and rendered IR, structural diff, flag every dropped bond and missing atom. No model in the loop.

edit
Model picks the op. Never the SVG.

Natural language → one of six safe rendering ops. The transform is fully deterministic; the verifier runs again after every edit.

Safety = the model doesn't get a pencil. It gets a multiple-choice quiz.

05 · Built today

What's real,
what's a stand-in.

Shipped

Chemistry end-to-end is real.
  • Real BANA six-dot tactile geometry — not a Unicode font
  • Deterministic verifier on rdkit-js canonical SMILES
  • Natural-language edit with Claude resolver + regex fallback
  • A4 print-sheet for swell paper + .brf for standard text embossers — runs on the kit schools already have
  • Serverless VLM proxy, key locked server-side
  • Public Vercel URL · QA-able on any device

Honest scope

What we are not pretending.
  • Pipeline parses chemistry only — other subjects upload as stand-in
  • NL edit limited to six safe rendering ops, not arbitrary structural change
  • No physical embosser or swell-paper heater on stage — we ship the file
  • The verifier announces its own ignorance instead of guessing
